# Hands on III – COVID-19 Severity Classification

This repository contains the analysis for **Hands on III (DMI course)**.  
The objective of the project was to classify COVID-19 patients as **non-Severe or Severe** based on their serum proteomic and metabolomic profiles using a Random Forest model.

The analysis is based on the dataset from the publication:  
**Proteomic and Metabolomic Characterization of COVID-19 Patient Sera**  
Shen et al., Cell (2020)

---

# Report

The full analysis report is available here:  
➡ **[Hands_on_III.html](Hands_on_III.html)**

The HTML report contains:
- Data preprocessing and quality control
- Random Forest classifier with cross-validation (Exercise 1)
- Feature importance and biological interpretation (Exercise 2)
- External validation on an independent test cohort (Exercise 3)

The report is fully self-contained and includes all figures.
